# MAOCS — My And Our ChatS

Веб-мессенджер с личными и групповыми чатами, звонками (WebRTC), загрузкой файлов в облако, друзьями и блокировками.

## Встановлення

1. Клонуйте репозиторій або розпакуйте архів.
2. Створіть віртуальне середовище (рекомендовано):
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```
3. Встановіть залежності:
   ```bash
   pip install -r requirements.txt
   ```

## Запуск локально

**Стандартний запуск (Flask + SocketIO):**
```bash
python app.py
```
Сервер буде доступний за адресою: **http://127.0.0.1:5000**

**Запуск через Gunicorn (production):**
```bash
gunicorn --worker-class eventlet -w 1 -b 0.0.0.0:5000 app:app
```

## Змінні середовища

| Змінна | Опис |
|-------|------|
| `PORT` | Порт сервера (за замовчуванням 5000 у `app.py`) |
| — | Секретний ключ зараз захардкоджений у `app.secret_key`; для продакшену краще винести в `os.environ.get('SECRET_KEY')` |

Файл бази даних створюється автоматично в корені проєкту: `chat_v3.db`.

## Структура проєкту

```
.
├── app.py              # Flask-додаток, API, сокети, WebRTC-сигнали
├── requirements.txt    # Залежності Python
├── config.yml          # Конфігурація (якщо використовується)
├── chat_v3.db          # SQLite БД (створюється при першому запуску)
├── static/
│   ├── favicon.ico     # Іконка вкладки (додайте свій файл)
│   ├── manifest.json   # PWA-маніфест
│   └── sw.js           # Service Worker
├── templates/
│   ├── login.html      # Сторінка входу/реєстрації
│   └── chat.html       # Головна сторінка мессенджера
├── tests/              # Автоматизовані тести (pytest)
│   └── test_api.py
└── README.md
```

## Тестування

Встановіть pytest та flask тест-клієнт (якщо ще не встановлено):
```bash
pip install pytest pytest-cov
```

Запуск тестів:
```bash
pytest tests/ -v
```

Запуск з покриттям:
```bash
pytest tests/ -v --cov=app --cov-report=term-missing
```

Тести покривають: логін/реєстрацію, заявки в друзі (`send_friend_req`), видалення групи (`delete_group`, у т.ч. неіснуюча група — очікується 404), завантаження файлу з моком зовнішнього API.

## Додатково

- **Звуки:** розблоковуються при першому кліку на сторінці чату.
- **Іконка вкладки:** покладіть файл `favicon.ico` у папку `static/`, щоб вона відображалась у вкладці браузера.
